# DeltaXY homepage sample direction

*Sample homepage concept based on the calm editorial feel of themindfulai.dev*
*March 31, 2026*

---

## Source inspiration from the blog

From `C:\source\blog` the current blog aesthetic already gives us a usable base:
- **serif + sans pairing** (`Crimson Text` + `Inter` in the current implementation)
- **muted sage-led themes** (`current`, `earth`, `tea`, `moss`)
- **paper-like backgrounds**
- **soft borders instead of loud UI chrome**
- **quiet shadows**
- **centered, literary header rhythm**
- **calm spacing and restrained motion**

For DeltaXY homepage, we can reuse the same spirit but make it more enterprise and proof-led.

---

## Recommended practical font choice

Because the blog already uses it successfully, the most practical option is:

- **Headings:** Crimson Text
- **Body / UI:** Inter

This keeps visual continuity with themindfulai.dev and avoids introducing a new font risk.

If later you want a stricter Japanese typographic influence, we can test Zen Old Mincho / Zen Kaku Gothic New. But for the first website pass, **Crimson Text + Inter is safer and already proven in your visual language**.

---

## Recommended palette for the homepage

Borrow from the blog’s `tea`, `earth`, and `moss` themes.

### Primary page palette
- **Background:** `#FAF7F2`
- **Card background:** `#FCFAF7`
- **Primary text:** `#3A3530`
- **Secondary text:** `#6B6358`
- **Accent:** `#8B7E6A`
- **Accent light:** `#B8AB96`
- **Border:** `#E9E4DC`
- **Soft hover tint:** `rgba(139, 126, 106, 0.06)`

This feels calm, warm, and long-term.

## Sample homepage layout

### 1. Top header
Very light, editorial, low-noise.

**Left:** DeltaXY wordmark  
**Center/right nav:** Solutions / Industries / Proof / Partners / About / Contact  
**Far right CTA:** Send 3 sample docs

**Style:**
- thin bottom border
- no filled dark nav bar
- plenty of horizontal space
- serif wordmark, sans nav

---

### 2. Hero

Centered, calm, spacious.

**Eyebrow**
Document intelligence for regulated work

**Headline**
Document workflows that feel under control

**Supporting copy**
DeltaXY designs and implements document-heavy workflows for aviation, compliance, and other review-heavy operations — with measurable accuracy, reviewable outputs, and production delivery in weeks.

**CTA**
Send 3 sample docs

**Proof line below**
95% extraction accuracy · 40+ hours saved per week · 10,000+ documents processed

**Visual treatment**
No dashboard collage.
Use one of these instead:
- an elegant proof panel with 3 metrics
- a single softly framed document-to-output flow
- a quiet abstract composition using paper blocks / lines / trace marks

The hero should feel like a deep breath, not a pitch deck.

---

### 3. Trust statement
A narrow band with one line:

**Built on real workflows, not demo-ware**

Then a supporting sentence in small text:
Designed for messy documents, human review, and regulated work where confidence matters.

---

### 4. Problem and relief section
Split layout. Left side pain. Right side calm solution.

**Left heading**
Why most document AI creates more work than it removes

**Left points**
- manual review keeps expanding
- edge cases break the workflow
- generic tools create new uncertainty

**Right heading**
What changes when the workflow is designed properly

**Right points**
- structured outputs
- reviewable evidence
- confidence where it matters
- less checking, less rework

---

### 5. Services section
Three vertically stacked blocks, not a generic 3-column SaaS grid.

#### Workflow diagnostic
We review one high-friction workflow and identify where time, errors, and rework are actually happening.

#### 3-file pilot
Send 3 sanitized files. We return structured output and show what a production workflow would require.

#### Production implementation
For workflows that prove out, we build the system with review logic, structured outputs, and measurable performance.

Each block should feel like an editorial card with:
- serif title
- compact body copy
- quiet CTA link

---

### 6. Proof section
This should be the visual anchor of the page.

Use one larger proof story and two smaller proof tiles.

#### Main proof story
**95% extraction accuracy**

DeltaXY built a production workflow for aviation leasing operations, reducing manual review and turning repeated extraction into a structured process with human-verifiable outputs.

Support with 3 quiet metrics:
- 40+ hours saved per week
- 200+ documents processed per month
- delivered in 6 weeks

#### Smaller proof tile 1
500+ DORA requirements mapped

#### Smaller proof tile 2
10,000+ documents processed across live work

Style:
- warm off-white cards
- thin borders
- no heavy shadows
- serif metric numbers are okay if done tastefully

---

### 7. Industries section
Three calm columns or staggered blocks.

- Aviation leasing
- Compliance and regulatory operations
- Audit-style review workflows

Keep copy short. Let spacing carry the calm.

---

### 8. How it works section
Simple 4-step line or vertical timeline.

1. Share a sample  
2. We assess fit  
3. Run a pilot  
4. Productionize when ready

This section should reduce fear.
It should feel easy and structured.

---

### 9. Partner section
A quiet section with more commercial clarity.

**Headline**
Offer document AI capability without building an internal AI team

**Body**
DeltaXY works with advisory, audit, and consulting partners on a white-label or subcontract basis.

**Bullets**
- you keep the client relationship
- we handle workflow implementation
- start with a fixed-fee pilot

---

### 10. Final CTA
Keep it very simple.

**Headline**
Start with 3 sample files

**Supporting line**
We’ll show what can be extracted, what needs review, and whether the workflow is worth implementing.

**CTA button**
Send 3 sample docs

---

## Sample microcopy style

### Button labels
Use:
- Send 3 sample docs
- See proof
- Request a diagnostic
- Discuss an implementation
- Partner with DeltaXY

Avoid:
- Get started
- Learn more
- Book demo
- Explore platform

---

## Motion style
Match the blog’s calm feel:
- 180–250ms transitions
- soft fades
- very light upward movement on hover
- no bounce
- no flashy entrances

---

## What to literally reuse from the blog

### Reuse directly
- serif/sans rhythm
- warm background system
- thin borders
- soft shadow values
- comfortable line-height
- high whitespace discipline
- understated link hover behavior

### Adapt for company site
- stronger proof sections
- more explicit CTA placement
- less article-like centered reading everywhere
- more structured sales flow
- more operational trust language
